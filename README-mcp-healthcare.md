# MCP Healthcare Servers

Medical-specific Model Context Protocol servers including FHIR, PubMed, Apple Mail, and GraphRAG Protocol integration.

## Features

- **FHIR Server**: HL7 FHIR resource management
- **PubMed Integration**: Medical literature search
- **Apple Mail**: Healthcare communication
- **GraphRAG Protocol**: Medical knowledge graphs

## Installation

```bash
npm install
# or
pip install -r requirements.txt
```
